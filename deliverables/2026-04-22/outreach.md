## Outreach: Developer Niche SaaS

**Kategorie:** Micro SaaS

### 1. LinkedIn-Nachricht

```
Hi! Ich bin Samuel und baue gerade ein Tool für Solo-Entwickler.

Kurze Frage: Wie gehst du aktuell mit [Entwickler brauchen fokussierte Tools für spezifische Pain P...] um?

Ich suche 5 Leute für ein kurzes Feedback-Gespräch (15 min). Kein Verkauf — ich will verstehen, ob das Problem real ist.

Hättest du Interesse?
```

*Personalisiere die erste Zeile mit etwas Spezifischem aus dem Profil der Person.*

### 2. Cold Email

**Betreff:** Kurze Frage: Wie löst ihr [Entwickler brauchen fokussierte Tools fü]?

```
Hi [Name],

ich bin Samuel und recherchiere gerade ein Problem, das viele Solo-Entwickler betrifft:

Entwickler brauchen fokussierte Tools für spezifische Pain Points in ihrem Workflow

Ich baue einen Prototyp und suche 5 Leute für ein kurzes Feedback-Gespräch (15 min, remote). Kein Sales-Pitch — ich will nur verstehen, ob und wie ihr das Problem habt.

Als Dankeschön: Früher Zugang zum Tool + Lifetime-Rabatt, wenn es was wird.

Hättest du 15 Minuten diese Woche?

Beste Grüße
Samuel
```

*Verwende den echten Firmennamen statt [Name]. Halte es unter 150 Wörtern.*

### 3. Discovery-Call Fragen

**Einstieg:**
- "Erzähl mir kurz: Was ist deine Rolle und was machst du täglich?"
- "Wie sieht dein typischer Workflow für [Entwickler brauchen fokussierte Tools für spezifis] aus?"

**Problem-Tiefe:**
- "Was nervt dich am meisten an deinem aktuellen Prozess?"
- "Wie viel Zeit verbringst du pro Woche damit?"
- "Hast du schon versucht, das Problem zu lösen? Was hast du probiert?"
- "Was würde passieren, wenn das Problem morgen gelöst wäre?"

**Lösung-Test:**
- "Wenn ich dir ein Tool zeige, das [Ein CLI-Tool oder API-Service der genau einen Dev-Workflow l] — wäre das interessant?"
- "Was müsste das Tool UNBEDINGT können?"
- "Was wäre ein absolutes No-Go?"

**Zahlungsbereitschaft:**
- "Zahlst du aktuell für Tools die dieses Problem lösen? Wenn ja, wie viel?"
- "Wenn das Tool genau das löst — wären 19-39€/Monat im Rahmen?"
- "Wer entscheidet bei euch über Tool-Käufe in dieser Preisklasse?"

**Abschluss:**
- "Darf ich dir den Prototyp zeigen, wenn er fertig ist?"
- "Kennst du 2-3 andere Leute mit dem gleichen Problem?"

### 4. LinkedIn Post (öffentlich)

```
Ich baue gerade ein Tool für Solo-Entwickler.

Das Problem: Entwickler brauchen fokussierte Tools für spezifische Pain Points in ihrem Workflow

Mein Ansatz: Ein CLI-Tool oder API-Service der genau einen Dev-Workflow löst (Changelog-Gen, PR-Review, Doc-Gen, Error-Triage).

Ich suche 5 Beta-Tester die genau dieses Problem haben.

👉 Kommentiere "Interessiert" oder schreib mir eine DM.

Kein Verkauf. Ich will validieren, ob das Problem echt ist — bevor ich wochenlang baue.

#buildinpublic #microsaas #ai
```

*Poste morgens 8-9 Uhr (DACH) für maximale Reichweite. Antworte auf JEDEN Kommentar.*

### 5. HackerNews Post

**Titel:** Show HN: Developer Niche SaaS – Ein CLI-Tool oder API-Service der genau einen Dev-Workflow l

```
Hi HN,

I'm building developer niche saas because entwickler brauchen fokussierte tools für spezifische pain points in ihrem workflow.

The idea: Ein CLI-Tool oder API-Service der genau einen Dev-Workflow löst (Changelog-Gen, PR-Review, Doc-Gen, Error-Triage).

Target users: Solo-Entwickler, kleine Dev-Teams (2-10), DevOps-Engineers

I'm looking for feedback from people who have this problem. Is this something you'd pay 19-39€/Monat for?

What am I missing? What would make this a must-have vs nice-to-have?
```

*HN posts sollten auf Englisch sein. Poste zu US-Morgen (15-17 Uhr MESZ). Antworte sachlich auf Kritik.*
